ตอนนี้เป็นการฝึกดจทย์ในห้องในเรื่องของ array ว่า array สามารถใส่ซ้อน array และตัว dict(object)
ใน array อีก เพื่อที่จะสามารถให้เราประมวลผลข้อมูลที่มีความซับซ้อนมากขึ้นอีกได้
และก็การเข้าถึงข้อมูลใน nested array และตัว object ใน array
เรื่องของ ... หรือ spread และ rest
1.กระจาย (spread) ค่าของarray และ object     ดึงค่าตัว array หรือ object มาเพิ่มใน array หรือ object อีกตัว 
ฟิลๆ เรียนรู้สกิลใหม่ในเกม
ex.
    const arr1 = [1, 2, 3];
    const arr2 = [...arr1, 4, 5]; // [1, 2, 3, 4, 5]
2.. Rest Parameter
ใช้ในฟังก์ชันเพื่อ รับค่าหลายค่าเป็นarray         รับ argument หลายตัวในฟังก์ชัน
ตอนแบบรับค่าหลายตัวมาทำไรต่อ
ex.
    function sum(...numbers) {
    return numbers.reduce((total, num) => total + num, 0);
    }
sum(1, 2, 3); // 6
